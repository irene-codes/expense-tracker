# Expense Tracker

A beginner-friendly Python expense tracking application with file storage.

## Features
- Add expenses with amount, date, and category
- View all recorded expenses
- Calculate total expenses
- Data saved to a file for persistence

## How to Use
1. Run the program: `python expense_tracker.py`
2. Choose from the menu:
   - Add expense
   - View expenses
   - Total expense
3. Follow the prompts to track your spending!
